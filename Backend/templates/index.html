Image To ASCII

Our ASCII Image API allows you to convert an image into ASCII characters and share the generated ASCII image together with the image path and an API key via our API interface. To do this, use a POST request with the image path and API key as parameters. The result is an ASCII character array that you can use in your code.



Visit api/img-to-ascii, and enter your api key + add an image path


curl -X POST -F "image=@path/to/your/image.png" -F "apiKey=[your-api-key]" https://image-to-ascii.up.railway.app/api/img-to-ascii
